  <br/>
  <br/>

## Installation

```
git clone https://github.com/Asam237/usefull-link
```

### API

```
cd api

cp .env.example .env

docker-compose build

docker-compose up -d

yarn install

yarn serve
```

### FRONTEND

```
cd frontend

yarn install

yarn dev

```
