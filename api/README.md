## Usefull Link App

Usefull Link API
<br/>

## Stack

- Node.js
- Express.js
- JsonWebToken
- Bcryptjs
  <br/>

## Installation

```
git clone https://github.com/Asam237/usefull-link

cd api

cp .env.example .env

docker-compose build

docker-compose up -d

yarn start
```
